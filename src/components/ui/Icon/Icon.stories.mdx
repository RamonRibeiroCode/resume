## Icons

Os ícones ajudam a criar páginas da web ilustrando conceitos e melhorando a navegação do site.
No entanto, o uso de ícones pode diminuir o desempenho da página.
Uma opção para evitar a diminuição do desempenho da página é usar SVGs de um
arquivo SVG único, localizado em `/public/icons.svg` e carregá-los com
o `components/ui/Icon`. Verifique a documentação do [Gerenciando ícones SVG](https://github.com/vtex-sites/nextjs.store#managing-svg-icons).

<br />

Esses são os sprites que utilizaremos dentro e fora do nosso front-end:

`/public/icons.svg`

Utilizaremos a ferramenta [SVGOMG](https://jakearchibald.github.io/svgomg/) para otimização.

<br />

---

Este projeto usa SVGs de <a ahref="https://remixicon.com/">Remix icons</a>.
